SAE32 : Développer des applications communicantes

Afin de mettre en pratique les compétences techniques que j’ai acquises durant mon troisième semestre avec JAVA. J’ai dû créer une application mobile sur Android.

Voir annexe

Objectif

L’objectif de ce projet est de développer une application Android sur Android Studio en JAVA qui reprend le concept de Twitter (X). L’application doit permettre de pouvoir envoyer un message et le liker. Les messages auront un certain ordre en fonction de leur date de publication et du nombre de likes. L’application doit être fonctionnelle et ergonomique.

Ma mission

Pour réaliser ce projet, j’ai commencé par créer une maquette sur Figma afin d’avoir une idée de quoi doit ressembler l’application.

Ensuite, j’ai commencé par créer une base de données sur Firebase (Service de Google) pour stocker les messages, les likes, les comptes utilisateurs. Occasion pour moi de pouvoir tester l’un des services les plus connus au monde.

Par ailleurs, j’ai mis en place le système de connexion/inscription afin d’identifier facilement un message. Cette partie, n’était pas forcément demandée mais j’ai décidé de la mettre en place pour coller avec la réalité d’une telle application.

Enfin, j’ai créé le cœur de l’application. Pour cela, je me suis organisé ainsi :

  • Publication d’un message par un utilisateur;
  • Permettre de pouvoir liker les messages de chaque utilisateur;
  • Appliquer l’algorithme qui permet l’ordonnancement des messages dans l’application.

Résultat

Au final, j’ai pu mettre en place une application Android qui reprend le concept de Twitter. J’ai pu créer une base de données sur Firebase pour stocker les messages, les likes, les comptes utilisateurs. J’ai pu mettre en place le système de connexion/inscription pour identifier facilement un message. J’ai pu créer le cœur de l’application en permettant la publication d’un message par un utilisateur, de pouvoir liker les messages de chaque utilisateur et d’appliquer l’algorithme qui permet l’ordonnancement des messages dans l’application. En termes humains, ce projet m’a permis d’améliorer ma capacité à être patient, organisé, persévérant et efficace.
